# Discussion Archive

已 Resolved 的议题归档目录。

- **命名**：`DISC-YYYYWww-NNN-<slug>.md`（议题号 + 短题目，文件名可直接反查议题号；由 `python tools/new_disc.py close "<slug>"` 自动生成）。
- **内容**：归档时整体迁移 `Discussion.md`（保留 Issue Header / Open Questions / Posts / Resolution），随后根目录文件从 `tools/templates/Discussion.template.md` 重置。
- **不要修改**：归档后的议题视为只读历史。如需重开，新建议题号并在 Header 写明 `Reopens: <旧议题号>`。
